How to Use

How to Use: Upload two attendance files to match records using intelligent name matching.

File Format Requirements

Master List Format (Excel .xlsx):

  • Column A: Email
  • Column B: Participant Name
  • Additional columns (Student ID, etc.)

Raw Attendance Format (CSV/Excel):

  • Name column
  • Session columns with P/A values

Generated File Structure:

  • Matched Sheet: Successfully matched records
  • Unmatched Raw Sheet: Records requiring manual matching
  • Summary Sheet: Template for uploading records to server
